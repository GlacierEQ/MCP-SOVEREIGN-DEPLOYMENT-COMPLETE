### Prerequisites
- Node.js 18+
- Smithery account with deployment access
- API keys for: OpenAI, Pinecone, Mem0, CourtListener
- Hawaii State Court eFiling access

### Installation
```bash
# Clone the repository
git clone https://github.com/GlacierEQ/MCP-SOVEREIGN-DEPLOYMENT-COMPLETE.git
cd MCP-SOVEREIGN-DEPLOYMENT-COMPLETE

# Install dependencies
npm install

# Configure environment variables
cp .env.example .env
# Edit .env with your API keys

# Start the MCP server
npm start
```

### Smithery Deployment
```bash
# Deploy to Smithery cloud
npm run smithery:deploy

# Monitor deployment status
npm run agents:status
```
